# Bilibertograma

Instrumento de aferição da **bilibertação existencial** (Liberologia) — domínio das energias conscienciais + independência econômico-financeira, a serviço da interassistência e da proéxis.

Página única (HTML estático) para preencher as 100 perguntas do bilibertograma e receber:

- **Índice Bilibertométrico** (0–100), com notas e percentuais por eixo (energética, financeira, propósito);
- **gráficos** por estágio (radar e barras);
- **análise automática** do gargalo bilibertológico e do próximo passo prioritário;
- **exportação em PDF** do resultado.

As respostas ficam apenas no navegador do usuário — nada é enviado a servidores.

## Como publicar (GitHub Pages)

1. Suba o `index.html` para este repositório.
2. **Settings → Pages →** Branch `main`, pasta `/ (root)` → **Save**.
3. Em ~1 minuto a página fica disponível em `https://SEU-USUARIO.github.io/NOME-DO-REPO`.

## Uso local

Basta abrir o `index.html` em qualquer navegador (precisa de internet para carregar as bibliotecas de gráfico/PDF via CDN).
